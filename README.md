# 나라 정보 앱

## 기술 스택

- typescript
- react
- redux
- redux-thunk
- redux-devtools-extension
- react-hook-form
- styled-components
- webpack
- eslint
- prettier
- .env

## 실행

- 개발 서버

```bash
npm run start
```

- 빌드

```bash
npm run build
```

---

`.env` 파일은 `.gitignore` 에 등록해야 하지만, 다른 컴퓨터에서 쉽게 테스트하기 위해 등록하지 않았습니다.
